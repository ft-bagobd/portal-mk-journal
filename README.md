# portal-mk-journal

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文档的集合存储与交付点。

## 项目简介

portal-mk-journal 旨在提供一种轻量、灵活的 HTML 页面归档方式。仓库中的每个文件均为自包含的 HTML 页面，可直接在浏览器中打开，或通过静态托管服务发布。适用于记录、展示、测试或分享各种独立的前端内容。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放主要 HTML 页面
├── assets/             # 公共资源（CSS、JS、图片等）
├── archive/            # 历史版本或已归档页面
└── README.md           # 本文件
```

- `pages/`：存放当前活跃的 HTML 页面，按需组织子目录。
- `assets/`：页面共用的样式、脚本、字体或图片资源。
- `archive/`：不再更新但保留的旧版本页面，便于回溯。

## 页面归档说明

- 每个 HTML 页面应保持独立，尽量不依赖外部资源（或仅依赖 `assets/` 下的文件）。
- 页面命名建议使用英文小写字母、数字与连字符，避免特殊字符。
- 如需引用其他页面，请使用相对路径。
- 归档页面可在 `archive/` 下按时间或版本建立子目录。

## 维护说明

- 欢迎提交 Pull Request 添加新页面或更新已有内容。
- 请确保新页面在主流浏览器中可正常渲染。
- 不建议在页面中包含跟踪脚本、广告或商业推广代码。
- 如发现页面无法正常显示，请通过 Issue 反馈。
